Theme System

Customize the appearance of your Hexactyl panel

Available Themes

Hexactyl comes with five built-in themes to match your preferences and branding:

Hexactyl (Default)

The classic Hexactyl theme with orange accents and dark backgrounds.

Midnight Blue

A deep blue theme for a professional, modern appearance.

Forest Green

An earthy green theme inspired by nature and forests.

Sunset Orange

A warm, vibrant theme with sunset-inspired colors.

Clean Light

A bright, clean light theme for daytime use.

Changing Themes

Switch between themes using the panel settings or configuration file:

Via Panel Settings

  1. Log in to your Hexactyl panel
  2. Click on your profile icon in the top right
  3. Select Settings
  4. Navigate to Appearance
  5. Choose your preferred theme from the dropdown
  6. Click Save Changes

Via Configuration File

Set the default theme for all users in the .env file:

# Theme Configuration
APP_THEME=hexactyl

# Available options:
# - hexactyl (default)
# - midnight-blue
# - forest-green
# - sunset-orange
# - clean-light

Via Admin Panel

Administrators can set the default theme for all users:

  1. Go to Admin Panel → Settings
  2. Navigate to Appearance
  3. Select the default theme
  4. Choose whether users can override the theme
  5. Save your settings

Theme Preview

Preview themes before applying them to see how they look:

Live Preview

The theme preview feature allows you to see changes in real-time:

  1. Open the Settings page
  2. Click on a theme card to preview it
  3. The panel will temporarily switch to show the theme
  4. Click Apply to save or Cancel to revert

Theme Differences

Each theme has unique characteristics:

  • Hexactyl: Orange (#f97316) accent, dark navy backgrounds
  • Midnight Blue: Blue (#3b82f6) accent, deep slate backgrounds
  • Forest Green: Green (#22c55e) accent, dark forest backgrounds
  • Sunset Orange: Burnt orange (#ea580c) accent, warm brown backgrounds
  • Clean Light: Blue (#2563eb) accent, white/light gray backgrounds

Custom CSS Support

Hexactyl supports custom CSS for advanced theme customization:

Adding Custom CSS

Create a custom CSS file to override default styles:

# Create custom CSS directory
mkdir -p /etc/hexactyl/custom/css

# Create your custom stylesheet
nano /etc/hexactyl/custom/css/custom.css

Custom CSS Example

/* Custom Hexactyl Styles */

/* Change primary color */
:root {
  --color-primary: #your-color-here;
  --color-primary-hover: #your-hover-color;
}

/* Custom sidebar styling */
.sidebar {
  background: linear-gradient(180deg, #1a1a2e 0%, #16213e 100%);
}

/* Custom button styles */
.btn-primary {
  border-radius: 8px;
  font-weight: 600;
}

Loading Custom CSS

Enable custom CSS in the configuration:

# Custom Theme Settings
CUSTOM_CSS_ENABLED=true
CUSTOM_CSS_PATH=/etc/hexactyl/custom/css/custom.css
⚠️ Important Notes: Custom CSS may break with panel updates. Always backup your customizations and test changes in a development environment first.

CSS Variable Reference

Hexactyl uses CSS variables for theming. Key variables include:

  • --color-primary - Main accent color
  • --color-primary-hover - Hover state color
  • --bg-primary - Main background color
  • --bg-secondary - Secondary background color
  • --text-primary - Primary text color
  • --text-secondary - Secondary text color
  • --border-color - Border color
  • --sidebar-bg - Sidebar background color
💡 Theme Development: To create a complete custom theme, fork an existing theme from the GitHub repository and modify the CSS variables.
