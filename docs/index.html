Complete guide to managing your game servers with Hexactyl

Welcome to Hexactyl

Hexactyl is a modern, feature-rich game server management panel designed to make server administration simple and efficient. Built with security and performance in mind, Hexactyl provides everything you need to manage multiple game servers from a single, intuitive interface.

Whether you're running a small Minecraft server for friends or managing a large network of game servers, Hexactyl scales to meet your needs. With support for multiple games, automatic backups, user management, and a powerful API, Hexactyl is the ultimate solution for game server administration.

Key Features

Hexactyl comes packed with features that make server management a breeze:

  • Multi-Game Support: Manage Minecraft, ARK, Rust, and many other game servers
  • User Management: Create users with different permission levels
  • Automated Backups: Schedule backups to local storage or S3-compatible services
  • Real-Time Monitoring: Track CPU, memory, and network usage
  • Console Access: Direct terminal access to your servers
  • File Management: Upload, download, and edit server files
  • Docker Integration: Isolated server environments for security
  • RESTful API: Automate server management with our comprehensive API

System Requirements

Before installing Hexactyl, ensure your system meets these minimum requirements:

Minimum Requirements

  • Operating System: Ubuntu 20.04+, Debian 11+, or CentOS 8+
  • RAM: 2GB minimum (4GB recommended)
  • Storage: 10GB free disk space
  • Docker: Docker 20.10+ and Docker Compose 2.0+
  • Network: Ports 80, 443, and 25565 (or your game server port) open

Recommended Specifications

  • CPU: 2+ cores
  • RAM: 8GB or more
  • Storage: SSD with 50GB+ free space
  • Network: 100Mbps+ connection

Getting Help

If you need assistance with Hexactyl, there are several ways to get help:

  • Documentation: Browse through these documentation pages for detailed guides
  • GitHub Issues: Report bugs or request features on our GitHub repository
  • Discord Community: Join our Discord server for real-time support
  • FAQ: Check our FAQ page for answers to common questions

When reporting issues, please include as much detail as possible, including your operating system, Docker version, and any error messages you've encountered.
