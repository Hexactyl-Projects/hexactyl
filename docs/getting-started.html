Getting Started

Your first steps with Hexactyl game server management panel

Requirements

Before you begin, ensure you have the following prerequisites met:

System Requirements

  • Operating System: Ubuntu 20.04 LTS or newer, Debian 11+, or CentOS 8+
  • Docker: Docker Engine 20.10+ and Docker Compose 2.0+
  • Memory: Minimum 2GB RAM (4GB recommended for production)
  • Storage: At least 10GB free disk space
  • Network: Stable internet connection with ports 80, 443 available

Software Dependencies

  • Git: For cloning the repository
  • curl: For downloading installation scripts
  • Basic Linux knowledge: Command line familiarity recommended
⚠️ Important Note: Hexactyl requires Docker to function properly. If you don't have Docker installed, please follow our Installation Guide first.

Quick Start Guide

Follow these steps to get Hexactyl running quickly on your system:

Step 1: Download Hexactyl

Open your terminal and clone the Hexactyl repository:

git clone https://github.com/hexactyl/hexactyl.git
cd hexactyl

Step 2: Configure Environment

Copy the example environment file and configure it:

cp .env.example .env
nano .env

At minimum, you should configure these variables:

  • PANEL_URL - Your panel's URL (e.g., https://panel.yourdomain.com)
  • APP_KEY - Generate with openssl rand -base64 32
  • DB_PASSWORD - Secure database password

Step 3: Start Hexactyl

Launch Hexactyl using Docker Compose:

docker compose up -d

Step 4: Access the Panel

Open your web browser and navigate to your panel URL. You'll be greeted with the setup wizard to create your admin account.

💡 Pro Tip: For production deployments, always use HTTPS. Configure a reverse proxy with Nginx or Apache for SSL termination.

First Steps After Installation

Once Hexactyl is running, complete these essential configuration steps:

1. Create Admin Account

During the initial setup, you'll create the primary administrator account. Choose a strong password and remember your credentials. This account will have full control over the panel.

2. Configure Server Node

Hexactyl uses a node-based architecture. After logging in:

  1. Navigate to Admin Panel → Nodes
  2. Click Create New Node
  3. Enter your server's details (name, FQDN, port)
  4. Copy the node token for the daemon installation

3. Install Daemon

The Hexactyl Daemon runs on your game servers and manages containers. Install it on each node:

curl -sSL https://get.hexactyl.dev | bash

4. Configure Game Servers

Create your first game server by:

  1. Going to Servers → Create New
  2. Selecting a game type (Minecraft, ARK, etc.)
  3. Choosing the node and allocation
  4. Setting resource limits (CPU, RAM, disk)

Creating Your First Server

Let's walk through creating your first game server in detail:

Server Creation Process

After setting up your node, creating a server is straightforward:

  1. Navigate to Servers: Click "Servers" in the main navigation
  2. Click Create: Press the "Create New Server" button
  3. Basic Details:
    • Server Name: A descriptive name for your server
    • Server Type: Select the game you want to host
    • Version: Choose the game version
  4. Allocation: Select the node and port allocation
  5. Resources: Set CPU, memory, and disk limits
  6. Environment: Configure game-specific settings
  7. Create: Click "Create Server" to deploy

Managing Your Server

Once created, you can manage your server from the server dashboard:

  • Console: Real-time server console with command input
  • Files: File manager for uploading and editing server files
  • Databases: Create and manage MySQL/PostgreSQL databases
  • Schedules: Set up automated tasks and restarts
  • Backups: Configure automatic backup schedules
  • Users: Grant access to other users
